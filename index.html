CarbonWatch Kenya

Transparent carbon credit flows and community voice for NRT conservancies in northern Kenya.

Tools

  • Community Interface — SMS and IVR for community input and alerts.
  • SCOUT Engine — Empowerment and governance surveys.
  • Public Dashboard — Project data, verification cycles, and conservancy map.
  • Sources — Data sources and methodology.

About

CarbonWatch connects NRT conservancy communities to carbon credit flows through communication, measurement, and public accountability. Community feedback flows via SMS/IVR; SCOUT maps inputs to empowerment dimensions (ILRI WELI). The public dashboard shows project-level data and per-conservancy figures where disclosed.

Verra project summary (VCS 1468)

Official data from the Verra Registry. Project status: On Hold — see notification letter on Verra.

  • ID: 1468 (VCS). Proponent: Northern Rangelands Trust. VCS / CCB status: On Hold.
  • Estimated annual emission reductions: 1,797,493 tCO2e. Total buffer pool credits: 1,238,467.
  • VCS project type: Agriculture Forestry and Other Land Use (AFOLU), ALM. Methodology: VM0032.
  • Area: 1,993,075 ha. Crediting period (1st): 01/01/2013 – 31/12/2042. Registration: 05/05/2020.
  • Validators: VCS — Ruby Canyon Engineering, Inc.; CCB — First: Aster Global Environmental Solutions Inc., Second: Ruby Canyon Engineering, Inc. CCB Third Edition.
  • Verra Registry VCS 1468
